Agents Detail

📖
@paper_reader
paper_reader.agent.md
論文 PDF を読み込み、要点を要約。summary.md を生成する。
💡
@hypothesis
hypothesis.agent.md
要約を元に改良仮説を複数提案する。
🔬
@experiment
experiment.agent.md
仮説を検証する実験計画（評価指標・比較対象・失敗条件）を作成する。
⚙️
@code_reviewer
code_reviewer.agent.md
コード改良案・最小差分の baseline scaffold を生成する。
✍️
@paper_writer
paper_writer.agent.md
実験結果から論文草案（Abstract / Intro / Method / Results / Discussion）を執筆。
🔄
research_workflow.py
research/research_workflow.py
論文 PDF から草案と code scaffold を自動生成する CLI ツール。
